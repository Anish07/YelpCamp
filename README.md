# YelpCamp

**YelpCamp** is a full-stack web application that allows users to create, review, and discover campgrounds. It features secure authentication, interactive maps, and image uploads.

-   **User Authentication**: Register and login with secure sessions (Passport.js).
-   **Campground Management**: Users can create, edit, and delete campgrounds.
-   **Reviews**: Users can rate and review campgrounds.
-   **Interactive Maps**: Campgrounds are displayed on a cluster map and individual location maps (Mapbox).
-   **Image Uploads**: Drag-and-drop image uploads (Cloudinary).
-   **Security**: Protection against common attacks (Mongo Injection, XSS, etc.).
